# Digital Garden

A minimal, thoughtful space for engineering essays, personal reflections, and daily journals. Built with [Jekyll](https://jekyllrb.com/).

## Getting Started

### Prerequisites
- Ruby (version 2.5.0 or higher)
- Bundler (`gem install bundler`)

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd digital-garden
    ```

2.  Install dependencies:
    ```bash
    bundle install
    ```

3.  Run the local server:
    ```bash
    bundle exec jekyll serve
    ```

4.  Open your browser at `http://localhost:4000`.

## Structure

- **`content/`**: All markdown content lives here, organized by type (`_essays`, `_engineering`, etc.).
- **`_layouts/`**: HTML templates for pages.
- **`assets/`**: CSS and other static assets.
- **`_config.yml`**: Site configuration.
